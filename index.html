Pyramid Counter Manual

Welcome to the Pyramid Counter app! This tool helps you do pyramid workouts with any exercise like squats, push-ups, pull-ups, or burpees.

Max Number: Set the maximum number for your pyramid (e.g., 10). The app will count up to this number, then back down.

Manual Mode: Tap the large button to go to the next number. The number color goes from green to red as it increases. Each tap represents one set of repetitions.

Auto Mode: Tap "Auto" to automatically progress through the pyramid. Each number has an exercise time and a rest time (based on the number). The main button shows the countdown during rest.

Sounds: - Manual mode plays a click sound.
- Auto mode plays a progress sound when the number changes.

Modes: Choose an exercise type from the dropdown. Custom mode allows you to adjust seconds per rep and rest per rep.

Enjoy your pyramid workout! Stay consistent and follow the numbers as they appear.
