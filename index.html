Pyramid Counter ‚Äî Help

This app guides you through pyramid sets. Set the Max number (top of your pyramid). Choose a mode (Squats, Push-ups, etc.).

Manual mode

  • Tap the big round button to start a step; the clock becomes a stopwatch.
  • When you finish reps for the step, tap the button again to stop stopwatch and start the next step's stopwatch (until you reach Max).
  • Number display above the button shows the repeats for the current step.

Auto mode

  • Tap Auto to automatically cycle through the pyramid (exercise + rest). The clock shows remaining time.
  • While Auto runs the big button is disabled to avoid accidental taps.
  • When Auto completes the pyramid (up and down), it will stop and reset (no loop).
